Dicas de estudo

Consuma outras mídias:

Com a facilidade de encontrar informações com a tecnologia moderna, muito da resolução de problemas depende da capacidade do desenvolvedor encontrar as respostas que procura ao invés de memorizar tudo de uma vez.

Leia artigos, ouça podcasts e consuma outras mídias de sua escolha além do curso praticado.

Anote:

Anote tudo o que precisar, anote tudo o que não precisar também. Melhor encontrar o que procura no futuro do que faltar por não achar necessário no momento.

Ensine os outros:

Ensinar as pessoas é uma ótima maneira de fixar na sua mente o conteúdo que você está ensinando e desenvolve suas capacidades de se relacionar com os outros que será útil em trabalhos em grupo.

Olhe outros códigos:

Programação é lógica, se o código perfeito é alcançado ele será sempre o mesmo, não há espaço para egoísmo. Claro que não simplesmente copie o trabalho dos outros, mas aprenda com as diferenças e evolua.

Foque em uma coisa de cada vez:

Existe muita informação disponível, mantenha o foco e procure apenas o que for relacionado as suas necessidades naquele momento. Isso evita perder 4 horas assistindo uma lista de vídeos sem fim que no final das contas não irá te ajudar no projeto que tem que entregar ainda hoje.

Lidando com grande problemas:

Analise o problema e começe a identificar blocos que podem ser tratados separadamente. Resolva uma coisa por vez ao invés de tentar abraçar tudo de uma vez.

Como iniciar um projeto

Programação envolve muito menos tempo realmente escrevendo código do que as pessoas imaginam. Faça e simplifique o seu processo de acordo com o necessário: desenhe, escreva no papel, pense em como montar suas idéias antes de começar direto no código para não se perder.

História da computação

Computação vem de computar: calcular, apurar, averiguar. Uma calculadora simplesmente cospe o resultado do processamento dos dados inseridos pelo usuário enquanto o ato de computar vai mais além e nos fornece informações adicionais que utilizamos no nosso planejamento. Em torno de 3000 AC surgiu o ábaco, uma das primeiras "calculadoras conhecidas" Levou em torno de 4800 anos para a humanidade desenvolver a capacidade de receber informações contextuais baseadas nos dados inseridos no século 19. O primeiro computador disponível comercialmente apareceu apenas por volta de 1950.

O que é um computador

O computador é uma máquina que, por meio de uma interface, processa os dados recebidos pelo usuário e retorna informações ao mesmo de acordo com as instruções dadas. O computador faz exatamente aquilo que lhe é designado, quando os resultados não estão de acordo com o que era esperado é, na maioria dos casos, falha humana, tando do usuário quanto do programador.

É composto de:

    Processador

Responsável por realizar os cálculos e operações lógicas necessárias.

    Placa mãe

Placa onde são ligados todos os outros componentes.

    Disco Rígido

Onde são armazenados os dados.

    Memória

Responsável por pegar os dados necessários no momento no disco rígido e mante-los em um estado onde o processador consegue utiliza-los

    Dispositivos de entrada e saída

São os equipamentos que fazem a interface entre o usuário e o computador em si, como teclado, mouse e monitor.

    Sistema Operacional

É o conjunto de programas que nos permitem interagir com o hardware do computador.

Sistema Operacional

O sistema operacional é responsável por fazer o gerenciamento do hardware e outros subsitemas, esses são os programas como por exemplo uma calculadora, um editor de texto, tudo que o usuário acessa para realizar uma atividade específica. Os sistemas operacionais mais conhecidos que temos para computadores são: Windows, Linux e Mac, e nos smartphones temos IOS e Android.

É composto de:

    Kernel

Kernel é a parte do sistema que gerencia o hardware em si posibilitando a operação do computador e mantendo seu funcionamento em níveis seguros como eletricidade e temperatura.

    Sistema de Arquivos

O sitema de arquivos é um conjundo de instruções lógicas que permite ao sistema operacional o acesso e controle sobre os dados gravados no disco rígido.

Algoritmos

Algoritmo é um conjunto finito de instruções ordenadas de forma lógica criado para resolver um problema. De maneira abstrata parece complicado mas não é. Uma receita de bolo, por exemplo é um algoritmo, um conjunto finito de instruções(acaba quando o bolo está pronto) que manipula os ingredientes para que o resultado seja o bolo pronto.

Estruturas

Array

Array é uma coleção de variáveis, como uma lista, onde cada valor é armazenado em uma posição do array começando do 0.





Objeto

Objetos são variáveis que possuem propriedades ligadas aos mesmos.





Array de Objetos

Um array de objetos é uma coleção de objetos. Cada objeto é salvo em uma posição do array e cada propriedade é ligada ao objeto.








Loops

Loops são empregados para percorrer posições de dados ou outras tarefas repetitivas.








Português estruturado

Português estruturado é um tipo de pseudocódigo, uma linguagem utilizada para aprender algoritmo e lógica de programação com sintaxe em português para manter o maximo de foco no aprendizado.

O que é a WEB?

WEB é uma sigla que significa World Wide Web (Rede Mundial de Computadores). Aplicação WEB é uma aplicação que "roda" na internet (navegador + servidor)

Ambiente de desenvolvimento

Para desenvolver aplicações WEB é necessário apenas um editor de texto e um navegador. Embora é possivel escrever código no bloco de notas e até no EDIT do MS-DOS (ambos são editores de texto.), no nosso caso utilizamos o VSCode que possui mais funcionalidades e auxilia na escrita do código e possui a possibilidade de instalação de extensões para aumentar o conforto e a produtividade de cada desenvolvedor de maneira individual.

Entendendo o CSS

CSS Cascading Style Sheets (Folhas de Estilo em Cascata) é uma linguagem de estilização (não programação). Ela altera o visual dos elementos da aplicação web marcados pelo HTML.

Principais propriedades do CSS

Color

Altera a cor da fonte








Background Color

Altera a cor de fundo do elemento








Width

Altera a largura de um elemento






Height

Altera a altura de um elemento






Margin

Altera o tamanho da área fora do elemento






Padding

Altera o tamanho da área interna do elemento






Border

Cria uma borda ao redor do elemento




Border Radius

Arredonda as bordas de um elemento




Font Size

Altera o tamanho da fonte




Font Weight

Altera a grossura da fonte




Background image

Insere e estiliza uma imagem de fundo dentro do elemento












Text Align

Faz o alinhamento de textos

Esconde ou torna visível um elemento mantendo seu lugar ocupado na página WEB

Altera o formato que o elemento ocupa na página

Cores no CSS

As cores no CSS podem ser declaradas pelo nome em inglês, pela propriedade RGB ou pela propriedade hexadecimal.

CSS inline ou folha de estilo externa

No CSS temos 3 maneiras de implementar o código:

Embedded é a tag style utilizada diretamente na tag alvo no arquivo HTML

Inline é uma tag style dentro do cabeçalho do HTML que abriga todas as linhas de código CSS utilizadas na página

Folha de estilo externa é um arquivo .CSS que contém exclusivamente os códigos CSS e é relacionada no cabeçalho do arquivo HTML

Hierarquia do CSS e suas exceções

O CSS é um arquivo de hierarquia em cascata com exceções, ou seja, ele executa os comandos por ordem de escrita o que causa os comandos no final do arquivo sobrescreverem os comandos anteriores.

Entre dois comandos que alteram o mesmo elemento, o comando mais recente (em baixo) é o que vale.

Exceções:

O CSS também cumpre com a hierarquia de acordo com o nível de nomeação dos elementos na seguinte órdem:

ID
DIV
CLASS

Ou seja, um comando atribuído à uma DIV e outro atribuído à uma CLASSE que alteram o mesmo elemento, o comando atribuído à DIV será o válido, indiferente da posição do comando dentro do arquivo

Existe também uma hierarquia "lateral" no CSS na seguinte ordem:

Embutido(dentro da própria tag HTML)

Inline (tag STYLE dentro do cabeçalho da página HTML)

Arquivo externo(styles.css por exemplo)

Quanto mais "próximo" da tag, maior a prioridade do comando CSS, o que dificulta as coisas quando algo dá errado pois um código embutido altera um elemento e não aparece no arquivo externo CSS. Se não existe um comentário mostrando o local do código embutido, o desenvolvedor pode perder muito tempo até encontra-lo

O ultimo nível de hierarquia é o comando "!important"

Este comando possui a prioridade mais alta em todo o CSS, quando ele é utilizado o CSS ignora toda a hierarquia e executa o comando que possui o "!important".

Se mais de um comando "!important" é utilizado no mesmo elemento toda a hierarquia volta a ser considerada, mas apenas entre os "!important"

Exemplo: CLASS !important e ID !important, o comando ID tem a prioridade.

Introdução ao JavaScript

JavaScript é a linguagem de programação escolhida para aplições WEB, ela é a responsável por todas as ações e manipulações de dados e memória na aplicação WEB

DOM (Document Object Model)

DOM (Modelo de Objeto de Documento) é a representação em objetos dos arquivos da aplicação web.

Enquanto os arquivos HTML e CSS são instruções em texto, a DOM é um espaço em memória que representa estes dados de uma maneira que pode ser manipulada pelas linguagems de programação.

Ao invés de declarar diretamente uma tag HTML por exemplo, na linguagem de programação, é feita a referência desta tag com uma váriavel com a ajuda do DOM. A partir deste ponto a linguagem de programação manipula apenas as variáveis declaradas.

Variáveis no JavaScript

VAR

VAR é o tipo básico de variável. Seu conteúdo é disponível globalmente(em todo o programa) e todas as VAR são criadas antes de qualquer outro código, independente do local onde foi declarada no arquivo.

LET

LET é um tipo de variável que mantém o valor declarado apenas dentro do escopo do bloco ou função onde foi declarada. Este tipo de variável é criada no local do código onde foi declarada.

CONST

CONST é um tipo de variável que após ser declarada só pode ser alterada manualmente pela edição de código, esta variável não pode ser alterada pelo programa em si. Útil em casos onde precisamos utilizar valores específicos como fórmulas matemáticas e dados de operação.

Exemplos: 3.14(pi), Valor da porcentagem de um desconto.

Tipos de dados

STRING Este tipo de dado trabalha com texto que é guardado em memória mas não é processado. Qualquer tipo de mistura de caracteres pode ser salvo no formato STRING.





NUMBER

Qualquer forma de números(inteiros, reais, negativos...) é considerado do tipo NUMBER. Este tipo de dado pode ser processado pelo programa.




BOOLEAN

Este tipo de dado trabalha apenas com duas possibilidades: TRUE(Verdadeiro) e FALSE(Falso). É utilizado para validação de condições












NULL

Este tipo de dado aparece somente quando uma operação inválida acontece.




UNDEFINED

Este tipo de dado aparece em todos os casos onde uma variável não tem um valor declarado




ARRAY

Array é um tipo de dado que guarda mais de um dado ao mesmo tempo. Os dados são separados de acordo com suas posições dentro do Array e podem ser armazenados dados de diversos tipos dentro do mesmo Array.




OBJETO

Objeto é parecido com um ARRAY onde guardamos primeiramente um dado e suas respectivas propriedades. Cada objeto é separado por sua posição e suas propriedades são ligadas ao objeto








FUNCTION

FUNCTION(função) é um tipo de dado que guarda um código para executar uma tarefa.





Devtools

Devtools é um conjunto de ferramentas para desenvolvimento e depuração WEB que existe em quase todos os navegadores hoje em dia. Basta clicar com o botão direito do mouse na tela do navegador e selecionar "Inspecionar".

Terminal

Terminal também conhecido como linhas de comando ou Console nos permite utilizar o computador sem uma interface gráfica de usuário, apenas com comandos de texto. Antes do Windows havia o MS-DOS (Hoje CMD). Também no Windows há o PowerShell. No Linux o terminal é chamado apenas de terminal.
